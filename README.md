## 博客功能

### page customization

1. 屏蔽掉默认的导航
2. 导航栏的样式自定义
3. 博客背景自定义
4. 公告栏文字样式自定义
5. 文章末尾签名样式自定义
6. 去除文章评论下的广告

### bulletin board

1. 添加我的博客链接 
2. 添加音乐
3. 添加公告栏图片并指向首页链接
4. 添加公告栏文字信息

### page heading

1. 添加博客顶部信息
2. 添加顶部滚动信息（公告）例子：考研祝福
3. 创建新的导航栏
4. 博客页面脚添加 `Github` 链接和博客链接

### page footer

1. 添加鼠标点击文字特效
2. 添加点击爆炸效果
3. 添加鼠标跟随效果
4. 添加雪花飘落效果
5. 添加看板娘 `visible record`
6. 添加文章目录
